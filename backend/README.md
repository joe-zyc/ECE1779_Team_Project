# Backend Template

This folder contains a backend scaffold derived from `IMPLEMENTATION_PLAN.md`.

## Included

- Express API entry points and route modules for the planned endpoints
- Placeholder controllers and middleware for the planned endpoints
- Shared response helpers and application error primitives

## Not Included

- Real endpoint logic
- Database queries
- Service and repository implementations
- JWT issuance and verification
- File upload implementation
- Notification delivery service integration

Every unfinished business endpoint currently returns `501 Not Implemented`.
